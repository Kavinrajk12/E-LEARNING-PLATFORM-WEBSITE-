# ChemLearn — IB Chemistry R2: Kinetics

An interactive, single-page study guide for **IB Chemistry R2: Kinetics** (Standard & Higher Level).

## 🚀 Live Demo

Open `index.html` directly in any browser — no build step required.

## 📚 Content Covered

| Section | Level | Topics |
|---|---|---|
| R2.1 — Collision Theory | SL + AHL | Rate factors, activation energy, Maxwell-Boltzmann |
| R2.2 — Rate Expressions | SL + AHL | Rate law, order, rate constant k |
| R2.3 — Integrated Rate Laws | AHL | First/second order, half-life |
| R2.4 — Arrhenius Equation | AHL | k = Ae^(−Eₐ/RT), ln k vs 1/T graph |
